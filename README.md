Gesior2012
==========

Gesior 2012 - Account Maker (website) for OTSes

Version for TFS 0.4 rev 3703+ engine ( https://github.com/otland/tfs-old-svn/tree/r3777 ).

Tested on PHP 7.0-7.4 and 8.0 with MariaDB.

PHP extensions required:
```
bcmath
dom
gd
json
mbstring
mysql
pdo
xml
```
